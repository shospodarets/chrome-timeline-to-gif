Convert a Chrome Timeline data to an animated GIF (video)

  1. Open "Timeline" Tab in Dev Tools ( Chrome browser )
  2. Click "Record"
  3. Do stuff in your page you want to see in result video
  4. Stop recording
  5. Right click anywhere on timeline data-> "Save Timeline Data..."
  6. Upload created file via "Choose File..." button or just drag-n-drop it in the browser window
